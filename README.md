# SoftGrowTech_Password_Generator
Built a Secure Password Generator using Python!
I developed a Python-based password generator that creates strong and random passwords with a mix of letters, numbers, and special characters.
🔧 Features:
✔️ User-defined password length
✔️ Includes uppercase & lowercase letters
✔️ Ensures at least one digit and one special character
✔️ Randomized and secure output
💡 What I learned:
📌 Python string & random modules
📌 Input validation & error handling
📌 Writing reusable functions
This project helped me strengthen my understanding of:
📌 Python basics
📌 Conditional statements
📌 Loops and data structures
